7. Animations

All animations use GSAP (ScrollTrigger, SplitText) or CSS keyframes. Lenis smooth scroll enabled on desktop only.

Neon Ignition
Words light up sequentially simulating neon tube ignition. Each word has unique timing: "Maluje sciany," ignites quickly with one hiccup, "ktore" catches smoothly, "opowiadaja" clean flash, "historie..." struggles with multiple failed attempts before catching. Outro follows same pattern.
GSAP timeline + ScrollTrigger (intro: center 66%, once: true) + IntersectionObserver (outro: threshold 0.3)
Card Expand (circle to rectangle)
First card image starts at scale(0.5) with clip-path: circle(closest-side). On scroll, scales to 1.0 and at 80% progress transitions from circle to full rectangle via expanding radius. Inner image counter-scales from 1.5 to 1.0.
GSAP ScrollTrigger onUpdate, clip-path manipulation, gsap.set()
Card Text Reveal (SplitText)
Title split into individual characters. Each char wrapped in overflow-hidden div with inner span. On reveal: chars slide in from x: 110% with staggered delay (0.015s). Description fades in from x: 40px, opacity: 0. Dim layer fades from 0 to 1.
GSAP SplitText + animateContentIn/animateContentOut functions
Marquee Scroll
Horizontal infinite scroll of Bebas Neue text phrases. Positioned at vertical center of first card. Fades out as card image expands (scale 0.5-0.75 range). Per-phrase hover highlight (mouse tracking via rAF throttle).
setupMarqueeAnimation() + CSS translateX + mousemove listener
Portfolio Card Transitions
Previous card scales down (0.75 desktop / 0.85 mobile) and fades out as portfolio section enters viewport. Portfolio wrapper similarly fades when next card arrives.
GSAP ScrollTrigger onUpdate with progress-based scale/opacity
Panorama Scroll
Three.js plane with panoramic texture. Camera pans horizontally on scroll progress. Three caption groups appear/disappear at segment boundaries (15%-75% within each third). Desktop: driven by CustomEvent from page.tsx pin. Mobile: own ScrollTrigger pin with scrub: 0.5.
Three.js + GSAP ScrollTrigger + SplitText captions
Image Scale on Entry
Non-intro card images start at scale(2.0) (desktop) or scale(1.05) (mobile) and scale down to 1.0 as the card scrolls from bottom to top of viewport.
GSAP ScrollTrigger onUpdate, gsap.set()
Portfolio Wiggle
On hover, portfolio images wiggle slightly: rotating between -0.3deg and 0.3deg with decreasing amplitude over 0.5s.
CSS @keyframes wiggle (tailwind.config.ts)
Navbar Entrance
Slides in from y: -100 to y: 0. Mobile menu icon rotates 90deg between hamburger and X. Mobile menu animates height from 0 to auto with opacity transition.
Framer Motion (motion.nav, AnimatePresence)
Loop Scroll / Tape
TrustedBy collaboration logos scroll infinitely via translateX(0%) to translateX(-50%) over 30s.
CSS @keyframes loop-scroll (tailwind.config.ts animation)
